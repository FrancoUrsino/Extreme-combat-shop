# INSTALACIÓN

**Para clonar el proyecto y tenerlo en tu pc local:**

```bash
  git clone https://github.com/FrancoUrsino/ProyectoFinal_Ursino
```

ir a la carpeta de tu proyecto

```bash
  cd my-project
```

instalar las dependencias

```bash
  npm install
```

iniciar en localhost

```bash
  npm start
```
